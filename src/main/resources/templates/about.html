ABOUT THIS PROJECT

Project Overview

This is a simple Java parser implementation that demonstrates the principles of lexical analysis, tokenization, and recursive descent parsing. It is capable of parsing and validating simple arithmetic expressions and creating a syntax tree representation.

The parser handles identifiers, basic arithmetic operations (+, -, *, /), parentheses for grouping, and memory operations (& for address, * for dereference).
